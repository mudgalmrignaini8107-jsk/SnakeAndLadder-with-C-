Snake and Ladder using C++

Overview
This is a Snake and Ladder game built using C++ as a console-based application. It simulates the classic board game 
with snakes, ladders, and dice rolls. The project helps understand basic game logic and C++ programming concepts.

1.Single Player Mode: he player competes against the computer in a turn-based system. Both roll a dice and move across
the board from start to finish. The first to reach the final position wins.
2.Two Player Mode: Two players play locally by taking alternate turns. Each player rolls the dice and moves forward on 
the same board. The first player to reach the end wins the game.

Features

1.Single Player Mode
Player competes against a computer-controlled opponent with turn-based dice rolls.
 First to reach or cross position 30 wins the game automatically.
 
2. Two Player Mode
 Two players play locally by taking alternate turns on the same board.
 Snakes and ladders affect both players equally, making the game competitive and dynamic.

 Getting Started
 
Prerequisites
A C++ compiler (e.g., GCC, Clang, MSVC)
Basic knowledge of C++ and console operations

Installation
Clone the Repository:

git clone https://github.com/mudgalmrignaini8107-jsk/SnakeAndLadder-with-C-.git

Compile the Code:
g++ main.cpp -o snake

Run the game:
./snake

How to Play

Choose Game Mode:
Start the game and select either Single Player or Two Player mode.

Single Player Mode:
Play against the computer and take turns rolling the dice until someone wins.

Two Player Mode:
Two players take alternate turns on the same device to reach the finish first.

Game Rules:
Roll the dice each turn, move forward by the number shown, and follow snakes (go down) and ladders (go up). 
The first player to reach or cross position 30 wins.
